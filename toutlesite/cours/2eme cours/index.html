Tous les documents HTML ont une structure requise qui comprend la déclaration suivante avec des éléments
d'ouverture et de fermeture:

<!DOCTYPE html>
<html>
<head>
<body>

La déclaration de type de document, ou <!DOCTYPE html>, informe les navigateurs Web de la version HTML utilisée. Il
est placé au tout début du document HTML. Après la déclaration du type de document, l'<html> élément marque le
début du document.


À l'intérieur de l'élément, l'<head>élément identifie le haut du document. Cependant, le contenu à l'intérieur de
l'<head>élément n'est pas affiché sur la page Web elle-même. Au lieu de cela, il doit inclure le titre du document, qui
est affiché sur la barre de titre dans la fenêtre du navigateur, des liens vers des fichiers externes ou toute autre information utile.
